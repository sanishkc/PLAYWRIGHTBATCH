# VS Code Keyboard Shortcuts (Windows)

A concise list of commonly used Visual Studio Code keyboard shortcuts on Windows.

- **Ctrl+Shift+P**: Show All Commands (Command Palette)
- **Ctrl+P**: Quick Open (Go to File)
- **Ctrl+N**: New File
- **Ctrl+O**: Open File
- **Ctrl+S**: Save
- **Ctrl+Shift+S**: Save As
- **Ctrl+W**: Close Editor
- **Ctrl+K Ctrl+S**: Open Keyboard Shortcuts
- **Ctrl+`**: Toggle Integrated Terminal
- **Ctrl+Shift+`**: Create New Integrated Terminal
- **Ctrl+B**: Toggle Sidebar Visibility
- **Ctrl+Shift+E**: Show Explorer
- **Ctrl+Shift+F**: Search in Files
- **Ctrl+F**: Find in File
- **Ctrl+H**: Replace in File
- **F3** / **Shift+F3**: Find Next / Find Previous
- **Ctrl+G**: Go to Line
- **Ctrl+Shift+O**: Go to Symbol in File
- **Ctrl+Space**: Trigger Suggest (IntelliSense)
- **Ctrl+/**: Toggle Line Comment
- **Shift+Alt+A**: Toggle Block Comment
- **Alt+Up / Alt+Down**: Move Line Up / Down
- **Shift+Alt+Up / Shift+Alt+Down**: Copy Line Up / Down
- **Ctrl+Shift+K**: Delete Line
- **Ctrl+Enter**: Insert Line Below
- **Ctrl+Shift+Enter**: Insert Line Above
- **Ctrl+L**: Select Line
- **Ctrl+D**: Add Selection To Next Find Match
- **Ctrl+Shift+L**: Select All Occurrences of Current Selection
- **Ctrl+Tab**: Switch Editor Tabs (recently used)
- **Ctrl+K Z**: Toggle Zen Mode

Tips:
- Use **Ctrl+K Ctrl+S** to customize shortcuts.
- Press **Ctrl+Shift+P** and type "keyboard" to open keyboard shortcut reference.
