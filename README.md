# PLAYWRIGHTBATCH

A small collection of JavaScript examples and notes used for learning Playwright and core JavaScript concepts.

## Contents

- Chapter_01_Basics: Basic JS examples
- Chapter_02_JavaScript_Concepts: Language concepts and examples
- Chapter_03_Identifier_Literals: Identifier and literal examples

## Quick start

Requirements:
- Node.js (for running the example .js files)

Run an example with:

```bash
node path/to/file.js
# e.g.
node Chapter_02_JavaScript_Concepts/06_Identifier_Rules.js
```
